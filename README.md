### `whoami`

Senior SDET with 10 years building test infrastructure for systems where reliability is non-negotiable — airline payments, multi-tenant healthcare SaaS, fintech, and financial market data. For the past several years, focused on **testing AI-powered features**: LLM evaluation harnesses, RAG retrieval testing, golden-dataset regression, and agentic system testing.

I build the frameworks, runners, and model-based test systems, and wire evaluation into CI/CD.

→ **[Full background, experience & competencies on mariusargatu.com/about](https://www.mariusargatu.com/about)**

### What I work on

- **AI / LLM testing** — Deepeval, RAGAS, Langfuse · faithfulness, answer relevancy, hallucination · RAG metrics (hit-rate, MRR, precision/recall@k) · agentic multi-turn & tool-call testing
- **Test architecture** — framework & runner design · model-based testing (xState) · property-based testing (Schemathesis, Hypothesis, fast-check) · Pydantic schemas
- **The thesis behind my writing** — *your evals are checks, not tests.* I write about the classical software-testing disciplines the LLM industry forgot it already solved.
